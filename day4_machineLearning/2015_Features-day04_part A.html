Using Features to describe different aspects of your image

The goal of this practical is to apply different derivative filters to an image to examine how this emphasises different structures within the image.

1a) Using Fiji open the image entitled: 'organoid.tif'. An image-stack should load with three frames. Split this stack into individual images using the Image->Stacks->'Stack to Images' command. We are going to work only on 'organoid-0001', close the other images.

1b) First of all we want to visualise different edge filters. Goto Plugins->FeatureJ->Feature Derivatives. These filters smooth the image and emphasis areas where intensity is changeing within an image. Start by setting 'X-order differentiation' and selecting the value '1.0', press 'Ok'. A new image should appear. Notice how certain parts of the image are more prominent than others. This is a '1st' derivative edge filter and highlights the edges which are moving up-to-down in the image.

1c) Try the Feature Derivatives plugin on the original image, but this time set the 'X-order differentiation' back to zero and set the 'Y-order differention' to one.

1d) Experiment with some of the other filters in the Plugins->FeatureJ menu.

1e) Choose your favourite filter response and use a Lookup-Table (Image->Lookup-Table) to colour it nicely.

When you perform machine learning for image segmentation an image is often described using many filters like the ones you have experimented with. Through using the pixel values in each filter output image, the machine learning algorithm is able to reproducibly classify the areas you are interested in, as long as you train it with some example data (You will do this later).

In addition to different types of filter, the feature description process also involves different scales of filtering. This generally means that your image is blurred prior to applying another filter. This has the effect of merging or blurring smaller high frequency details together. This means when you filter your image you should draw out larger (or broader) structures giving a sense of scale.

1f) Try applying the 'X-order differentiation' filter again but with different smoothing. Make a small figure which has '1.0', '2.0', '4.0' and '8.0' smoothing on the same image. Pick a Lookup-Table which helps visualise the difference in the images.
