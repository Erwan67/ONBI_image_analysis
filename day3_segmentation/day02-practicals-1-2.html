General Segmentation

1) Watershed Segmentation (40 min)

Fiji provides an implementation of the Watershed segmentation method we discussed in class. Use the following instructions and the information in the online tutorial to segment the two following two images to try out the Watershed segmentation. 

Use the following steps to run to segment the images:

  1. Download the images and load them into Fiji
  2. Apply a Gaussian filter to smooth the images 
  3. Use the thresholding function in Fiji (Image->Adjust->Threshold) to create a binary image
  4. Invert the image (Edit->Invert)
  5. Run the Watershed method (Process->Binary->Watershed)
  6. Run the particle statistics to identify the different objects (Analyze->Analyze Particles)

Try to change the parameters for the linear filter in Step 2 and see what overall effect this has. You can also try a median filter to perform the de-noising. 

Complete the following tasks:

  • Describe in your own words what the Watershed method (as implemented in Fiji) does. 
  • Not all the cells are being segmented perfectly. What failure modes do you observe?. 

 

2) Refining Segmentations Using Morphological Filters (20 min)

Morphological filters can be used to refine the binary mask you generate in the first exercise. You find the list of morphological filters under Process->Binary. Try the erode and dilate functions to see if you can improve the results you obtained in the previous exercise.

Briefly summarise your results. Compare the results with your efforts to improve the de-noising in the previous exercise.
